# Configuration Live Chat Crisp

## Étapes pour activer le Live Chat

1. **Créer un compte Crisp** (gratuit)
   - Aller sur https://crisp.chat
   - Créer un compte gratuit
   - Noter votre `WEBSITE_ID` (visible dans les paramètres)

2. **Mettre à jour le fichier**
   - Ouvrir `src/components/common/LiveChat.jsx`
   - Remplacer `YOUR_WEBSITE_ID` par votre vraie ID Crisp

3. **Le chat apparaîtra automatiquement** en bas à droite du site

## Fonctionnalités incluses

- ✅ Chat en direct
- ✅ Messages hors ligne (email)
- ✅ Notifications de nouveaux messages
- ✅ Historique des conversations
- ✅ Support multilingue
- ✅ Intégration mobile

## Personnalisation

Vous pouvez personnaliser le chat depuis votre dashboard Crisp :
- Couleurs, position, langue
- Messages automatiques
- Disponibilité
- Et plus encore...
